Facial Performance

— Animator / Modeler

Maya / Motion Capture

April 2015

Collaborating with Vidya Vinnakota, we created Facial Performance. This project experiments with mapping human features and expressions onto nonhuman characters modeled in Maya. Facial motion capture data was extracted from actors' faces and imported into Maya. Careful selections from the collected data gave us more control over which human features coresponded to features on the creatures.
